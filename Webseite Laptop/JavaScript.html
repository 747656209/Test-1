Anfang und Schluss

Am anfang und am schluss eines JavaScripts schreibt mann immer

Zwischen die Script kapitel schreibt mann dann das Javascript.

Befehle

alert Befehl

Der alert befehl sendet eine Meldung an den Benutzer.
In der Klammer und den anfürungszeichen
ist der Text der angezeigt werden soll.
Mann kann aber auch ohne Anführungszeichen den namen
einer Variable schreiben, dann wird der Text der Variable angezeigt

let und const Befehl

Mit dem let Befehl kannst du variablen bestimmen,
die im Script noch geändert werden können.

Mit dem const Befehl kannst du auch Variablen bestimmen,
aber die können im Script nicht mehr geändert werden.

prompt Befehl

Mit dem prompt Befehl kann man ein Eingabefenster öfnen,
mit dem mann einer Variable den Wert zuweisen oder ändern kann.
In der Klammer und den anfürungszeichen ist der Name des Eingabefensters.

if, else if und else Befehl.

Mit dem if befehl kannst du eine Bedingung erstellen,
die bedingung ist immer in den ().
Wenn diese erfüllt ist wird alles in den {} ausgeführt.

alle bedingungen die hinzugefügt werden müssen mit else if
bezeichnet werden.

Am ende der Bedingungen muss else hin,
das wird dann ausgeführt wenn keine bedingung erfüllt wurde.
